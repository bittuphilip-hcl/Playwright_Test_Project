**pre-requiste**: 
1. Latest version of Node-js and npm should be installed in your local machine.
2. run npx playwright install command to download the browsers


Install the Dependencies:
1. To install all the dependencies from package.json file run the command "**npm install**"
2. To Execute the scenarios, run the command
**npx cucumber-js**   or **npm run test**

3. To Execute the scenarios using tags, run the command
   **npm run test --TAGS="@TagName"**
